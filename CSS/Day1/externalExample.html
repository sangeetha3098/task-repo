External CSS

External CSS is used to style HTML webpages by writing CSS code in a separate .css file and linking that file to the HTML document. This helps keep HTML and CSS separate and makes website styling easier to manage.

Points About External CSS

  1. External CSS is written in a separate .css file.
  2. It is connected to HTML using the link tag
  3. The link tag is placed inside the head section.
  4. One CSS file can style multiple webpages
  5. It keeps HTML and CSS separate.
  6. Makes code clean and organized.
  7. Reduces repeated styling code.
  8. Preferred for large and professional websites.

Advantages

It can be reused in many webpage and easy to maintain , cleaner code.
